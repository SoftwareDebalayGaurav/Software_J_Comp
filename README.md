# Software_J_Comp
UML TO JAVA
========================

The project is a desktop based application where according to user inputs for class diagrams a Java code will get generated.


Dependencies
------------
- JDK 1.8
- JAVA EE 8


Libraries Used
------------
- Swing
- Windows Builder


Tested on Windows,Mac and Linux .

Modules 
---------
- Class Module
- Method Module
- Generate Module
